## BDD 

Utilisateur 
- nom 
- prenom 
- email
- type : eleve, prof, client 

Projet
- nom
- description
- (utilisateurs) 

Promo

- date de début
- date de fin

## Créer une BDD avec les install scripts de Daishi (Creation BDD + User dedié)

./mkdb.sh [app_name]
Créer une base de données avec le nom specifié + un user avec le nom specifié
